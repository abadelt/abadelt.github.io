Java EE 8 Status

A Panel Discussion with:

Ed Burns, Sebastian Daschner, David Delabassee, Ivar Grimstad, Werner Keil, Gunnar Morling, Anatole Tresch

moderated by Andreas Badelt

Ab brief history of Java EE 8

Early 2014: "Community Survey"

Official Start in Aug 2014, with new MVC JSR

Mid-2016: Most JSRs "on hold"

JavaOne 2016: Announcement of new direction and new survey

Early 2017: Survey Results support new direction

EE 8 gaining momentum again, JMS, Mgmt API & MVC out

Feb 2017: MVC handed over to Ivar Grimstad and "the community"

Planned Release: July 2017

JSR 365 - CDI 2.0

http://cdi-spec.org

Status: Approved

Highlights:

  • Supports Java SE
  • Asynchronous Events
  • Observer Ordering

Community Contributions: http://cdi-spec.org/contribute

Mailing List: http://lists.jboss.org/mailman/listinfo/cdi-dev

JSR 370 - JAX-RS 2.1

https://jcp.org/en/jsr/detail?id=370 / https://jax-rs-spec.java.net

Status: EDR

Highlights:

  • Server Sent Events (SSE)
  • Reactive client extensions
  • Non-blocking IO
  • Further alignment withing Java EE umbrella
